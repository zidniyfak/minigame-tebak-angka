# Tebak Angka - Minimalist

Selamat datang di proyek "Tebak Angka - Minimalist"! Ini adalah permainan tebak angka sederhana di mana Anda akan mencoba mengalahkan komputer.

## Deskripsi Proyek

Permainan ini adalah implementasi sederhana dari permainan tebak angka. Pemain memilih salah satu angka dari 1 hingga 5, dan kemudian komputer juga akan memilih angka secara acak. Hasilnya akan ditampilkan, apakah pemain menang, kalah, atau seri.

## Fitur

- **Pilihan Angka**: Pemain dapat memilih angka dari 1 hingga 5.
- **Melawan Komputer**: Bermain melawan komputer yang memilih angka secara acak.
- **Antarmuka Pengguna Responsif**: Dibuat dengan Bootstrap 5 untuk tampilan yang responsif dan menarik.
- **Notifikasi Interaktif**: Menggunakan SweetAlert2 untuk menampilkan hasil permainan dan notifikasi lainnya.
- **Animasi Menarik**: Menggunakan Animate.css untuk efek visual yang dinamis.
- **Efek Kemenangan**: Menggunakan Canvas Confetti untuk merayakan kemenangan pemain.

## Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web standar dan beberapa pustaka pihak ketiga:

- **HTML5**: Struktur dasar halaman web.
- **CSS3**: Penataan gaya kustom (`style.css`).
- **JavaScript**: Logika permainan (`script.js`).
- **Bootstrap 5**: Kerangka kerja CSS untuk desain responsif.
- **Google Fonts**: Font 'Poppins' untuk tipografi yang menarik.
- **SweetAlert2**: Pustaka JavaScript untuk pop-up yang indah dan responsif.
- **Animate.css**: Pustaka CSS untuk animasi siap pakai.
- **Canvas Confetti**: Pustaka JavaScript untuk efek confetti.

## Cara Menjalankan Proyek

Ada dua cara untuk menjalankan permainan ini:

### 1. Akses Langsung (GitHub Pages)

Anda bisa mencoba permainan ini secara langsung melalui GitHub Pages di:
https://zidniyfak.github.io/minigame-tebak-angka/

### 2. Menjalankan Secara Lokal

Untuk menjalankan permainan ini di lingkungan lokal Anda, ikuti langkah-langkah berikut:

1.  **Kloning Repositori**:
    ```bash
    git clone https://github.com/zidniyfak/minigame-tebak-angka.git
    cd minigame-tebak-angka
    ```
2.  **Buka `index.html`**: Cukup buka file `index.html` di browser web pilihan Anda (misalnya, Chrome, Firefox, Edge).

Anda sekarang siap untuk bermain "Tebak Angka"!
